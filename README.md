# Fibulacci

# Requirements

- NodeJS: min v16.15.1
- Yarn: min 1.22.19

# Clone

```bash
git clone https://github.com/kindelia/kindelia-js
git clone https://github.com/kindelia/fibullaci
```

# Install

```bash
cd fibullaci && yarn install && yarn build
cd ../kindelia-js && yarn install
```

# Run

```bash
yarn dev
```
