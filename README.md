# Fibulacci

# Requirements

- NodeJS: min v16.15.1
- Yarn: min 1.22.19

# Clone
```bash
git clone git@github.com:Kindelia/kindelia-js.git
git clone git@github.com:Kindelia/fibulacci.git
```

# Install
```bash
cd kindelia-js && yarn install && yarn build && cd ..
cd fibulacci && yarn install
```

# Run

```bash
yarn dev
```
